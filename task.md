Lộ trình học salesforce
Tuần 1: Càn quét Admin & Flow (Nền tảng hệ thống)
Đừng coi thường bước này, vì Developer giỏi là người biết khi nào không cần code.
Mục tiêu: Hiểu Object, Security (Profile/Permission Set) và Flow Builder.
Thực hành: Tạo một App quản lý dự án đơn giản. Dùng Flow để tự động gửi Email khi Task hoàn thành.
Tài liệu: Chạy ngay module Admin Beginner trên Salesforce Trailhead.
Tuần 2: Apex & SOQL (Backend - "Java-like")
Vì bạn biết JS, bạn sẽ thấy Apex hơi khắt khe về kiểu dữ liệu (Strongly typed).
Mục tiêu: Viết Trigger, Class, và truy vấn dữ liệu bằng SOQL.
Thực hành: Viết một Trigger ngăn không cho xóa "Account" nếu có "Contact" liên quan.
Tài liệu: Đọc nhanh Apex Developer Guide và làm các bài tập trên ApexSandbox.io để luyện tư duy logic.
Tu tuần 3: Lightning Web Components - LWC (Frontend)
Đây là "đất diễn" của bạn. LWC dựa trên chuẩn Web Components hiện đại.
Mục tiêu: Data binding, Event handling, và gọi Apex từ JS.
Thực hành: Xây dựng một Dashboard tùy chỉnh hiển thị biểu đồ doanh thu bằng JS library (như Chart.js) tích hợp vào Salesforce.
Tài liệu: Xem LWC Recipes trên GitHub để copy-paste và học cấu trúc code mẫu.
Tuần 4: Luyện đề & Thực chiến AI
Ôn thi: Truy cập Focus on Force mua bộ đề Platform Developer I. Cày nát 500+ câu hỏi để hiểu các "bẫy" về giới hạn hệ thống (Governor Limits).
AI Update: Học cách viết Prompt trong Prompt Builder để tích hợp AI vào code.
Chứng chỉ: Đăng ký thi Salesforce Certified Platform Developer I vào cuối tuần này.
Kỷ luật "thép":
Sáng (8h - 12h): Học lý thuyết trên Trailhead.
Chiều (13h - 18h): Code thực tế trên Developer Edition (free).
Tối (20h - 22h): Luyện đề trắc nghiệm.
