## üí° Tips & Best Practices

### Study Tips
1. **Consistency over intensity** - Study 2-3 hours daily rather than cramming
2. **Hands-on practice** - Always code along with tutorials
3. **Take notes** - Document what you learn
4. **Teach others** - Explaining concepts reinforces learning
5. **Review regularly** - Spaced repetition improves retention

### Exam Tips
1. **Read questions carefully** - Watch for "NOT" and "EXCEPT"
2. **Eliminate wrong answers** - Process of elimination
3. **Trust first instinct** - Don't overthink
4. **Manage time** - Don't spend too long on one question
5. **Answer all questions** - No penalty for wrong answers

### Coding Tips
1. **Follow naming conventions** - Be consistent
2. **Write test classes** - Always aim for 75%+ coverage
3. **Bulkify your code** - Avoid SOQL/DML in loops
4. **Use comments** - Explain complex logic
5. **Refactor regularly** - Improve code quality
